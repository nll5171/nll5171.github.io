Heritage for the Future: Alessi Mode

This was a romhack created in the summer of 2019 in order to replicate the Alessi mode present in the Dreamcast release of JoJo’s Bizarre Adventure: Heritage for the Future. The playable character Alessi has a move that allows him to transform characters into weaker versions of themselves temporarily (typically this transforms them into children, but not exclusively). Alessi mode has both characters fight in this state for the entire duration of the match.

The changes themselves were fairly simple:

  1. Have Alessi mode transformation begin at roundstart
  2. Remove the Alessi mode transformation timer, so that it never ends
  3. Reduce the number of rounds to always be 1

Each of these changes only required a byte or two to be changed, with the main challenge being finding the location of what needed to be changed.

When a match is initialized, a set of RAM addresses for each player are set to a default value of 0. By changing one of these RAM addresses to store the value that triggers the Alessi mode transformation, I was able to get the first, and most important, part completed.

After that, I used Final Burn Alpha’s RAM watch tool to identify what RAM address stored the Alessi mode timer. After that, I went into Mame Debugger and removed the check that would return the player to normal once initiated.

Finally,I repeated that process for the rounds required to win, as this is 2 by default and can be adjusted. Once done, I hardcoded the value to always be 1.
