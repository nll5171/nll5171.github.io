Blink

Blink was the first project I worked on in RITâ€™s Game Design & Development program. The game was created in a group of 5 as part of the Game Development & Algorithmic Problem Solving II class I took during the latter half of my first academic year. As this was our first project, this was not created using a game engine, but rather in Monogame, a C# framework for game development.

It is a puzzle game that sees the player control an eyeball that must carry boxes to specific destinations in each level. All the while, rats will be traveling along a specific path before targeting the eyeball once the path has finished. Shining light on the rats will return them to their starting position on their path, but your light is a limited resource that recovers slowly, and cannot be used while carrying boxes.

Given the fairly simplistic nature of the game, we decided to expand upon what we had through the use of an interactive tutorial. The player is able to play around several tutorial sections with text teaching them about each aspect of the gameplay. The first section for instance, is dedicated to movement and controls, while the final one explains enemy behavior, visually demonstrating it all the while for the player to experience.

My contributions to the project overall were with regards to everything concerning the light mechanics, from ensuring collision was in a proper cone shape and rotated accordingly with the position of the player. I also handled some of the process concerning the boxes, such as accommodating for rotation around the eyeball, and did code cleanup and documentation wherever possible.
